Dev Terminology
  • Function

    • In JavaScript, a Function is a block of code that performs a specific task or set of tasks and can be called or invoked by other parts of the program.

  • Let

    • Let is a keyword used in JavaScript for creating variables that can be changed later on in the program. You can use let to declare a variable that only exists within a certain part of your code, like a function or a loop, and cannot be accessed from outside that part of your code.

  • Const

    • Const is a keyword in JavaScript used for creating variables that cannot be reassigned a new value once they are initialized. It creates a read-only variable.

  • Div

    • In HTML, Div is a tag used to create a container or a block-level element that can be used to group other HTML elements together and apply styles or layout to them as a group. It is a commonly used tag in HTML for creating layout structures and organizing content.

  • Fieldset

    • In HTML, Fieldset is a tag used to group related form elements together and create a box around them with an optional legend. It's a semantic tag that helps in structuring and styling of forms.

  • Legend

    • In HTML, Legend is a tag used inside the Fieldset tag to provide a caption or title for the group of form elements enclosed within the Fieldset.
